# Skills

A collection of custom skills for Claude Code.

## Available skills

- [`skills/deck-from-template`](skills/deck-from-template) — Interview-driven PowerPoint builder that turns a conversation into a finished deck rendered on a brand/internal template.
- [`skills/jira`](skills/jira) — Take actions against a Jira Cloud instance: create/read/update/search issues, transition status, comment, manage assignees, attach files, and discover projects/issue types/fields.
- [`skills/discover-and-document`](skills/discover-and-document) — Explore an unfamiliar or existing codebase and generate grounded functional and technical documentation for it, regardless of language or framework.
